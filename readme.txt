This plugin increases the shopping experience of users by providing them enhanced shop and checkout page in WooCommerce Stores.

== Description ==

Pingu for WooCommerce is a plugin created by <a href="http://www.techpingu.com/">TechPingu</a> for WooCommerce websites to enhance their functionalities of shop and checkout page to create a better shopping experience. For using this plugin WooCommerce is required.

Main features of this plugin are:
<ul>
<li>Automatic Address Fields from Google Maps</li>
<li>Add Multiple Addresses</li>
<li>Easily Change Login from Checkout Page</li>
<li>Email and Mobile Login Options</li>
<li>Automatic Price Updating</li>
<li>Delete Address, Add Primary Address</li>
<li>One Page for Enhancing Shopping Experience</li>
<li>Customized Functionalities, Works with Any Theme and Plugin</li>
<li>Options to Change Colors, Texts and More Things in Shop and Checkout Page from WordPress Backend</li>
<li>No Coding Required</li>
<li>One in Only Plugin You Need for WooCommerce</li>
</ul>

For the premium version of this plugin you will have to buy it from <a href="http://products.techpingu.com/product/pingu-for-woocommerce/">here</a>.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `Pingu For Woocommerce` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Buy the premium version of plugin from http://products.techpingu.com/product/pingu-for-woocommerce/ 
5. Configure the plugin by going to the Pingu for WooCommerce Submenu in WooCommerce Menu
